Create a QR Code Generator using Google Colab

In this tutorial, we will create a QR code generator using Google Colab. Google Colab is a cloud-based platform that allows you to write and execute Python code in a Jupyter notebook-like environment.

Prerequisites

Before we get started, you will need the following:

  • A Google account
  • Access to Google Colab

Step 1: Setting up Google Colab

First, open Google Colab by visiting https://colab.research.google.com/. Sign in with your Google account if you are not already logged in.

  1. Click on "New Notebook" to create a new Colab notebook.

# Import the required library
import qrcode

# Generate a QR code object
qr = qrcode.QRCode(
    version=1,
    error_correction=qrcode.constants.ERROR_CORRECT_L,
    box_size=10,
    border=4,
)

# Add data to the QR code
qr.add_data(input("Enter the link or text:"))
qr.make(fit=True)

# Create an image from the QR code data
img = qr.make_image(fill_color="black", back_color="white")

# Save the QR code image
img.save("my_qr_code.png")
